HTML5 Game of the Week: Faerie Alchemy

We’re going to start spotlighting one HTML5 game each week, starting things off with Faerie Alchemy by Subsoap. We discuss the game in more detail in the ‘cast, but the gist is that it’s a solid HTML5 game, very appropriate for the platform and quite polished. Check it out!
